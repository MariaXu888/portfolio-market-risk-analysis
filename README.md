# Portfolio Market Risk Analysis (VaR & CVaR)

This project implements a Python-based portfolio market risk framework using SPY, TLT, and GLD ETF data.

## Methods Implemented
- Parametric Value at Risk (VaR)
- Historical Simulation VaR
- Monte Carlo Simulation VaR
- Conditional Value at Risk (CVaR / Expected Shortfall)

## Key Features
- API-based market data retrieval
- Log return modeling
- Covariance-based portfolio volatility estimation
- Multivariate Monte Carlo simulation
- Tail risk analysis

## Tools
Python, NumPy, Pandas, SciPy, yfinance

## Objective
To replicate practical portfolio risk measurement techniques commonly applied in trading and market risk environments.
